# Note

This is a prototype!

# Materials

- 4x M2 Nuts
- 4x M2 8mm deep Screws
- Wood Glue
- Sand Paper

# Instructions

1. Laser cut v1_case_bottom and v1_case_top
2. Glue pieces together
3. (Optional) Apply light sanding
4. Press nuts into larger diameter holes
5. Screw everything together.
